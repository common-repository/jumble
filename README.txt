'Sentence jumble' game for primary school.

== Description ==

It is a simple plugin to implement 'jumble sentence' style game for primary school. The aim of the game is to reorder scrambled words to make
a complete sentence. 
Game is triggered using a shortcode, [jumble sentence="this|is|a|sentence"]. The sentence is splitted at vertical bars and the fragments are shuffled.

== Installation ==

Just load the plugin.
